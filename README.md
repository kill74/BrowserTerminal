#  WebTerm v2.0 - O Teu Terminal Web Inteligente

Bem-vindo ao **WebTerm**! Se gostas da eficiência da linha de comandos mas não queres abdicar das funcionalidades do browser moderno, este projeto é para ti. É um terminal completo que corre diretamente no teu navegador, permitindo-te navegar na web, ver vídeos, ouvir música e até falar com IAs locais, tudo sem tirar as mãos do teclado.

---

##  Como Começar

Basta clicares no terminal e começares a escrever. Se te sentires perdido, escreve `help` e prime **Enter**.

### Atalhos de Teclado Úteis:
*   **`Tab`**: Completa automaticamente os comandos.
*   **`Setas Cima/Baixo`**: Navega pelo histórico de comandos.
*   **`Ctrl + R`**: Pesquisa reversa no histórico.
*   **`Enter`**: Executa o comando.

---

## Comandos Disponíveis

### Navegação e Pesquisa
*   **`search <texto>`**: Pesquisa na web.
*   **`engine <nome>`**: Muda o motor de busca.
*   **`browse <url>`**: Abre um site dentro do terminal.
*   **`curl <url>`**: Mostra o código HTML puro de uma página.

### Multimédia
*   **`youtube <pesquisa>`**: Procura vídeos no YouTube.
*   **`watch <url ou ID>`**: Vê um vídeo do YouTube.
*   **`twitch <canal>`**: Vê um stream da Twitch.
*   **`download <url>`**: Converte e descarrega um vídeo do YouTube para MP3.

### Sistema de Ficheiros Virtual (VFS)
*   **`ls`**: Lista ficheiros e pastas com tamanho e data de criação.
*   **`touch <ficheiro>`**: Cria um ficheiro.
*   **`mkdir <pasta>`**: Cria uma nova pasta.
*   **`cat <ficheiro>`**: Lê o conteúdo de um ficheiro.
*   **`rm <ficheiro>`**: Apaga um ficheiro ou pasta.
*   **`edit <ficheiro>`**: Abre o editor de código.
*   **`run <ficheiro.js>`**: Executa código JavaScript.
*   **`find <query>`**: Procura ficheiros pelo nome ou conteúdo.

### Integração GitHub
*   **`github <username>`**: Mostra o perfil do GitHub.
*   **`git clone <url>`**: Clona um repositório GitHub para o VFS.

### Inteligência Artificial (Ollama)
*   **`ai <pergunta>`**: Fala com a tua IA local.
*   **`ollama status`**: Vê o estado do Ollama.
*   **`ollama model <nome>`**: Muda o modelo da IA.

###Personalização e Utilitários
*   **`theme <modo>`**: Muda o visual.
*   **`css <estilos>`**: Injeta estilos CSS.
*   **`help`**, **`clear`**, **`date`**, **`echo <texto>`**, **`cd <pasta>`**.
